# Team Profile Images

This directory should contain the profile images for the "Meet our Experts" and "Meet the Founders" sections.
The website will automatically look for images in this folder with the format `firstname-lastname.jpg`.

If an image is missing or fails to load, the website will automatically display the person's initials on a branded background.

## How to Add Images
1. Save the LinkedIn profile picture (or any professional headshot) to your computer.
2. Rename the file to match the list above exactly.
3. Drag and drop the files into this folder.
4. Refresh the website.
